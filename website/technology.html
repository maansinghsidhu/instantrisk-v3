Security

Enterprise-Grade Protection

Insurance data demands the highest security standards. InstantRisk is built with defense-in-depth across every layer, from network perimeter to application logic.

  • ✓ AWS WAF — Web Application Firewall with managed rule sets for OWASP Top 10, SQL injection, XSS, and custom rate limiting.
  • ✓ End-to-End Encryption — TLS 1.3 in transit, AES-256 at rest. All data encrypted in RDS, S3, EFS, and ElastiCache.
  • ✓ VPC Isolation — Private subnets for backend services, database, and vector search. No direct internet access to sensitive components.
  • ✓ IAM & JWT Auth — Role-based access control with JWT tokens. Underwriter, analyst, and admin roles with granular permissions.
  • ✓ CloudWatch & Alarms — Full audit logging, security event monitoring, CPU/memory alerts, and automated incident response.
  • ✓ Automated Backups — Daily database snapshots, S3 versioning, and EFS backups with 30-day retention. RPO under 1 hour.
